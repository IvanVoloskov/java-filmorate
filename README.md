# java-filmorate
Template repository for Filmorate project.
### ссылка на диаграмму (https://dbdiagram.io/d/Filmorate-6949a55661d95806cf3948d0) 

## Краткое описания структуры БД
#### Таблица films ссылается на mpa_ratings, потому что каждому фильму должен быть присвоен один возрастной рейтинг.   
#### Связь films и genres реализована через таблицу film_genres, так как один фильм может относиться к нескольким жанрам, 
#### а один жанр — присваиваться множеству фильмов (связь многие-ко-многим).
#### Таблица likes связывает users и films, потому что она фиксирует, какой пользователь поставил лайк какому фильму,
#### и делает это с помощью связи многие-ко-многим, так как один пользователь может лайкнуть много фильмов,
#### а один фильм — получить лайки от многих пользователей.
#### Таблица friendships должна связывать таблицу users саму с собой,
#### потому что дружба — это отношение между двумя разными пользователями внутри системы.
